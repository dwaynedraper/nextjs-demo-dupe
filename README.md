# Next.js Demo

### The stack:

- Next.js
- Storybook
- Bootstrap 5
- React-Icons

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# production mode
npm run build
npm start
```

Next.js runs on localhost:3000 by default

## Running storybook

```bash
npm run storybook
```

Storybook runs on localhost:6006
